Reinforcement Learning

Learning by trying.

An agent is dropped into a world it doesn't understand. No instructions, no map — just what it can sense right now, the moves it can make, and a reward when things go well. By trying, failing, and trying again, it figures out a winning strategy on its own.

For the next few minutes, that agent is you.

Learning from you

The agent learns from your round — watch it play

Every step you took became one piece of experience: “in this spot, I tried this move, and got this reward.” The agent treats your moves as its exploration, and the reward is the signal that says which of those moves were good. It then plays the same board with what it learned — right after you.

How does that actually train a policy?

For each of your moves, the agent does a tiny update: nudge the value of that move in that spot toward reward you got + the value of where it led. A spike's −25 pushes those moves down; the goal's +80 pulls the moves that led there up; and crucially the value propagates backward step by step, so even the early moves on a good path become ‘good.’ This is Q-learning, and because it learns the value of the best move (not the one you happened to take), the agent can end up playing better than you did.

No demonstrations needed

Now watch an agent learn it from scratch
